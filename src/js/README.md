# 📏 Luftlinie — Entfernungsrechner

Berechne die direkte Luftlinienentfernung zwischen zwei Orten weltweit.

**Live:** https://rintuchowdory.github.io/luftlinie

## Features
- 🔍 Ortssuche mit Autocomplete (OpenStreetMap Nominatim)
- 📏 Haversine-Formel — direkt im Browser
- 🗺️ Interaktive Karte mit gestrichelter Linie
- ✈️ Geschätzte Flug- & Fahrzeit
- 🧭 Kompassrichtung & Zeitzonen-Differenz
- ◎ GPS-Standort Unterstützung
- 📱 Mobile responsive
- **Keine API-Key nötig!**
